KEY METRICS & SPECIFICATIONS
================================================================================

Color Palette:
  - Primary Accent: #4f46e5 (indigo)
  - Neutral Background: #f8f9fa (light mode), #1a1a2e (dark mode)
  - Success: #15803d on #dcfce7
  - Warning: #7c2d12 on #fef3c7
  - Error: #b91c1c on #fee2e2
  - Info: #1e40af on #eff6ff

Contrast Ratios (All WCAG AA+):
  - Primary text on page: 20:1 ✓ AAA
  - Secondary text on page: 7.5:1 ✓ AA
  - Buttons on white: 8.5:1 ✓ AA
  - All action colors: 7.5:1+ ✓ AA

Typography:
  - Font Stack: System fonts (-apple-system, 'Segoe UI', sans-serif)
  - Scale: 0.75rem to 3.157rem (modular)
  - Line Height: 1.2-1.6 for readability
  - Weights: 400, 500, 600, 700

Spacing:
  - Base Unit: 8px (0.5rem)
  - Scale: xs (4px) to 5xl (96px)
  - Standard Padding: 1rem, 1.5rem, 2rem
  - Grid Gap: 0.75rem to 2rem responsive

Responsive:
  - Mobile First Approach
  - Touch Targets: 44x44px minimum
  - Breakpoints: 640px, 1024px, 1440px
  - Adaptive Typography & Spacing

Animation Timing:
  - Fast: 0.15s (immediate feedback)
  - Normal: 0.3s (standard transitions)
  - Slow: 0.6s (celebrations)
  - Easing: cubic-bezier functions included
